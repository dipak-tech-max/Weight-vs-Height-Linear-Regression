# Weight-vs-Height-Linear-Regression
# 📏 Weight vs Height Prediction (Machine Learning)

## 🚀 Overview

This project predicts a person's **height based on their weight** using a Machine Learning regression model.

## 🧠 Model Used

* Linear Regression

## 📊 Dataset

* Contains weight and corresponding height values
* Cleaned and preprocessed before training

## ⚙️ Tech Stack

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib

## 📈 How It Works

1. Load dataset
2. Train regression model
3. Predict height from given weight

## ▶️ Run Locally

```bash
git clone https://github.com/your-username/Weight-vs-Height-Linear-Regression.git
cd Weight-vs-Height-Linear-Regression
pip install -r requirements.txt
python train.py
```

## 📌 Example

Input:
Weight = 70 kg

Output:
Predicted Height ≈ 170 cm
